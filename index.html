Example array 
Array ( [0] => Array ( [0] => 1 [1] => 1 ) [1] => Array ( [0] => 1 [1] => 1 ) )


This array is fed as example inside php as: 
$array[0][0]=1;
$array[0][1]=1;  
$array[1][0]=1;  
$array[1][1]=1;


This function can be called anywhere inside other documents as 
echo array2table($array);

11
